About TruthLens AI

Our mission, technology, and commitment to news transparency

Our Mission

TruthLens AI exists to fight misinformation by giving every citizen the tools to verify news before sharing it. In an era of viral falsehoods and echo chambers, we believe informed readers are the best defense against the spread of fake news.

We combine keyword analysis, source credibility scoring, and corpus matching to produce fast, explainable verdicts — with full transparency about how each conclusion was reached.

Technology Stack

  • Spring Boot 3.2 — MVC + REST API backend
  • MySQL — Trusted article corpus database
  • Bootstrap 5 — Responsive UI components
  • Thymeleaf — Server-side template engine
  • JavaScript Fetch API — Async REST communication
  • Spring Data JPA — Repository pattern with Hibernate

How the Verification Engine Works

1
Keyword Extraction

Submitted text is tokenized and stop-words are filtered. Meaningful unigrams and bigrams are extracted as search terms.

2
Category Detection

Signal keywords map the content to one of five categories: International, National, Politics, Sports, or Technology.

3
Source Retrieval

Up to 10 candidate articles are retrieved from the trusted corpus using full-text keyword queries.

4
Similarity Scoring

Each candidate is scored on keyword overlap, title match weight, content depth, and source credibility.

5
Verdict Derivation

Scores are aggregated into a confidence percentage. Thresholds determine Likely True, Likely False, or Needs Verification.

6
Results & References

The result is saved with matched source references and a human-readable explanation for full transparency.

Future Roadmap

The codebase is architected for easy external API integration:

  • NewsAPI.org — Live global news ingestion
  • Google Fact Check Tools API — Crowd-sourced fact checks
  • OpenAI GPT — Semantic similarity and explanation
  • Tesseract / Google Vision — Server-side OCR for images
  • Reuters / AP Wire — Real-time trusted source feeds

API keys can be added to application.properties without changing the core service layer.

Trusted Source Network

Our reference corpus draws from the world's most credible news organizations:

Reuters Associated Press BBC News The Guardian NPR Al Jazeera ABC News ESPN TechCrunch Wired

Credibility scores (0–100) are assigned to each source based on editorial standards, fact-checking policies, and correction transparency.
